##TCP Server (C++)

Simple TCP server written in C++ using low-level Linux sockets.


## 🚀 Features

• Interactive menu 1-4, 0.

• Show client IP and port.

• Shows your message (3).

• Send simple HTML response.


## ⚙️  Requirements

• Linux / macOS.

• g++ compiler.


## 💡 Build and Run

```bash
g++ main.cpp -o server
```

```bash
./server
```

```bash
Connect to server : nc 127.0.0.1 8080
```
